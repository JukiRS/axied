# The Nether

## Генерация через датапак Insendium

🔥 Адский мир генерируется с использованием датапака Insendium, который добавляет:

* большое разнообразие биомов;
* уникальные структуры;
* необычное снаряжение с эффектами (например, щит, испускающий волну радиации при блокировании удара).
