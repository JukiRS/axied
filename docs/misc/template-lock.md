# Блокировка шаблонов

🔒 Механика, позволяющая закрепить шаблоны за собой с помощью команды `/lock`.

## Как это работает

* Командой `/lock` можно **заблокировать шаблоны**.
* После блокировки **другие игроки не смогут заблокировать** эти шаблоны.
* Игрок, который заблокировал шаблоны, может **в любой момент разблокировать** их.
* Другие игроки разблокировать чужие шаблоны **не могут**.
